# Customer Churn Analysis (EDA Project)

## 📌 Project Overview
This project is focused on Exploratory Data Analysis (EDA) of customer churn data. The goal is to understand why customers are leaving and identify important patterns.

## 📂 Files in this Repository
- project1.ipynb → Jupyter Notebook with full analysis
- Customer Churn.csv → Dataset used for analysis

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Key Steps Performed
- Data cleaning and preprocessing
- Handling missing values
- Exploratory Data Analysis (EDA)
- Data visualization
- Identifying factors affecting churn

## 📈 Outcome
This analysis helps in understanding customer behavior and factors influencing churn, which can help businesses improve customer retention strategies.
